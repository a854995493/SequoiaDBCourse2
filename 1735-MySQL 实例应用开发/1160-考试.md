## 考试介绍

此考试是在完成部署 SequoiaDB 巨杉数据库引擎的环境中使用 MySQL 实例完成对数据进行操作、查询，熟练掌握使用 Java 进行 MySQL 开发。

#### 知识点

1. 创建数据表插入数据；
2. 基于 SequoiaDB 巨杉数据库引擎 MySQL 实例的 CRUD 操作；
3. 视图、常用函数的使用；
